environment variables:

```.env
EMAIL="yourInaturalistEmailHere"
PASSWORD="yourInaturalistPasswordHere"
INATSECRET="yourInaturalistSecretHere"
```

To run the crawler locally, ensure that the email and password in the .env is valid for an iNaturalist account and is within ```/in```\
Additionally, a secret is required to grab data using the iNaturalist api.\
For an example see the example.env.
