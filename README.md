# Aves and the Effects of Urbanization

--- 

## About

--- 

Hello! We are an undergraduate group at Louisiana State University conducting research on aves. The focus of our research is to understand bird patterns with regard to human population density. We will be analyzing bird counts and bird types as parameters and understanding how urbanization affects these parameters.

## Coding stuff

--- 

environment variables:

```
EMAIL="yourInaturalistEmailHere"
PASSWORD="yourInaturalistPasswordHere"
INATSECRET="yourInaturalistSecretHere"
```

To run the crawler locally, ensure that the email and password in the .env is valid for an iNaturalist account and is within ```/in```\
Additionally, a secret is required to grab data using the iNaturalist api.\
For an example see the example.env.
